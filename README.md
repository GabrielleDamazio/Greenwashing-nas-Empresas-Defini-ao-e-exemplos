# Greenwashing-nas-Empresas-Defini-ao-e-exemplos
Esta pesquisa tem como objetivo analisar o conceito de greenwashing. O estudo foi desenvolvido na disciplina de Inovação, Sustentabilidade e Competitividade Empresarial, buscando compreender como essa prática pode afetar a credibilidade das organizações, a confiança dos consumidores e a competitividade no mercado.
